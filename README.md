JQuery Methods Used: 

.replaceWith() - Used to dynamically swap out the main pet image element with a new image displaying different reactions (like happy, sad, or sleeping) when interacting with the buttons, and later reverting it back to the default image.

.after() - Used to insert the comic-style text speech bubble into the DOM immediately following an invisible anchor element, ensuring the message displays perfectly positioned above the pet image.
